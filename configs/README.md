# Configs

Каталог зарезервирован для конфигураций, которые не должны находиться в корне проекта.

Сейчас основные инструменты Python читают конфиг из `pyproject.toml` в корне, поэтому перенос не выполнялся для сохранения стандартного поведения экосистемы.

План:
- Выносить в `configs/` только дополнительные runtime-конфиги (например, шаблоны proxy/nginx, override-файлы compose, env presets).

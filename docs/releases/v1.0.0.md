# v1.0.0

Первый официальный релиз `silero-tts-api`.

## Что вошло в релиз
- Стабильный OpenAI-совместимый TTS API (`/health`, `/v1/models`, `/v1/audio/speech`)
- Полный набор проверок качества (`lint`, `typecheck`, `tests`, проверочная сборка Docker)
- Воспроизводимый локальный процесс: `make setup` и `make check`
- Базовые проверки безопасности зависимостей через `pip-audit`
- Обновленная документация и зафиксированные архитектурные решения

## Проверки
- `make check` выполнен успешно
- `make audit-deps` выполнен успешно
- Быстрая проверочная CPU-сборка Docker выполнена успешно

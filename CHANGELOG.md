# Журнал изменений

В этом файле фиксируются ключевые изменения по релизам.

## [1.0.0] - 2026-02-15

Первый публичный релиз `silero-tts-api`.

### Что включено
- Единый процесс разработки через `Makefile` (`setup`, `check`, `audit-deps`, `build`).
- Контур качества: `ruff`, `mypy`, `pytest`, `pre-commit`.
- Документация по архитектуре и процессам: ADR, аудит, план рефакторинга, структура репозитория, правила контрибьюта.
- Обязательные проверки CI с матрицей Python (3.10/3.12) и быстрой проверочной сборкой Docker.

### Безопасность
- Добавлены проверки уязвимостей зависимостей (`pip-audit`) для контуров выполнения, тестирования и Docker.
